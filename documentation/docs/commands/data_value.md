# Data Values

Data value are block/item variation. Data value can be rotation of the block, block colour, item durability, etc.

## List of Data Values

### Coloured Block

Colour     | Data Value
-----------|-----------
White      | `0`
Orange     | `1`
Magenta    | `2`
Light Blue | `3`
Yellow     | `4`
Lime       | `5`
Pink       | `6`
Gray       | `7`
Light Gray | `8`
Cyan       | `9`
Purple     | `10`
Blue       | `11`
Brown      | `12`
Green      | `13`
Red        | `14`
Black      | `15`
